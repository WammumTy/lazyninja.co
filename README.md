# Lazy Ninja

Welcome to the Lazy Ninja project! This repository contains the source code and resources for the **Lazy Ninja** website.

## Project Overview

Lazy Ninja is a platform designed to provide users with tools, tips, and resources to simplify their daily tasks and boost productivity.

## Features

- **User-Friendly Interface**: Easy to navigate and use.
- **Productivity Tools**: A collection of tools to help you stay organized.
- **Resource Library**: Access to guides, templates, and more.

## Getting Started

1. npm run build:dev

2. npm run preveiw

3. Continue Developing!
